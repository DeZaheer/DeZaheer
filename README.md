### 🛡️ [File Integrity Monitor](https://github.com/DeZaheer/file-integrity-monitor)
Python security tool that monitors a folder for unauthorized changes
by generating SHA256 hash fingerprints for every file and comparing
them against a saved baseline — similar to how enterprise tools like
Tripwire work in hospitals and banks.

**Key Features**
- Takes a baseline snapshot of all monitored files
- Generates unique SHA256 hash fingerprints per file
- Detects modified, deleted and newly added files
- Reports all changes with timestamps

**Skills Demonstrated**
Python · Cryptographic Hashing · File System Monitoring · Threat Detection

---

### 🔐 [Caesar Cipher Encryption Tool](https://github.com/DeZaheer/caesar-cipher)
Python tool that encrypts and decrypts messages using the Caesar
Cipher method — one of the oldest encryption techniques in history.
Includes a brute force decoder that demonstrates why weak encryption
is vulnerable to cryptanalysis.

**Key Features**
- Encrypts messages using a chosen shift key
- Decrypts messages with the correct shift key
- Brute force decoder that tries all 25 possible shifts
- Demonstrates the fundamentals of cryptography

**Skills Demonstrated**
Python · Cryptography · Encryption · Cryptanalysis
